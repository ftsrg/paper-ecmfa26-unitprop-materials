# paper-ecmfa26-unitprop-materials
Materials for the paper "From Constraints to Commands: Graph Predicate Differentiation in 4-Valued First-Order Logic"

The repository contains the following materials:
- `input/`: The input files used in the experiments
    - `generation-problems/`: The problems used for evaluating the model generation
    - `partial-models/`: The partial models used in the experiments for evaluating the model uncertainty reduction
- `results/`: The raw results obtained from the experiments
    - `generation-results.csv`: The results from the model generation experiments
    - `uncertainty-results.csv`: The results from the uncertainty reduction experiments
